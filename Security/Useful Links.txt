--- Malware, URL & File Analysis ---

Virus Total
https://www.virustotal.com/gui/home/upload
Allows security teams to submit files for analysis across dozens of antivirus engines and threat intelligence sources to quickly assess whether a file is malicious.

URL Scan 
URLScan.io
https://urlscan.io
Scans and analyzes websites to identify malicious behavior, phishing, and hosting infrastructure.

ANY.RUN
https://any.run
Interactive malware sandbox that allows analysts to execute and observe suspicious files and URLs in real time.

Hybrid Analysis
https://www.hybrid-analysis.com
Malware analysis service combining static and dynamic analysis to understand file behavior and indicators.

--- Early Detection ---

Canary Tokens
https://www.canarytokens.org/nest/
create lightweight deception artifacts (digital tripwires) that trigger alerts when accessed, indicating potential unauthorized access or attacker activity.

--- Network, IP & Infrastructure Intelligence ---

Shodan
https://www.shodan.io
Search engine for internet‑exposed devices and services, useful for attack surface discovery and infrastructure reconnaissance.

Censys
https://search.censys.io
Internet scanning and asset discovery platform used to identify exposed hosts, certificates, and services.

IP Tracker
https://www.ip-tracker.org/
public IP intelligence tool that provides geolocation, ISP, ASN, and DNS details for IP addresses.

Arin
https://search.arin.net/rdap/
investigate suspicious IPs, validate network ownership during incident response, and support threat hunting, abuse reporting, and attribution workflows.

Abuse ip Database
https://www.abuseipdb.com
Community‑driven database for reporting and checking malicious IP addresses, frequently used for quick reputation checks during triage.

--- Domain & Registration Intelligence --- 

Whois
https://www.whois.com/whois/
provides public registration details for domains and IP addresses, including ownership, registrar, nameservers, and registration dates. 

--- Credential & Breach Intelligence ---

Have I Been Pwned
https://haveibeenpwned.com
Breach data lookup service used to identify compromised credentials during incident response and investigations.

CIS Controls
https://www.cisecurity.org/controls

Mitre 
https://attack.mitre.org/
knowledge base that documents real‑world adversary tactics and techniques based on observed cyber attacks.
